Chart

  • Home
  • Chart

Membuat chart tidak jauh berbeda dengan dokumentasi resmi chart js, namun kamu perlu tag div atau section dengan class .chart, didalamnya ada dua bagian yaitu Chart Header (tempat judul dari chart) dan Chart Body (tempat chart), untuk membuat Chart Header kamu perlu tag div dengan class .chart__header, dan letakkan judul dari chart didalamnya menggunakan tag h6. Untuk membuat Chart Body kamu perlu tag div dengan class .chart__body, letakkan Chart Body dibawah Chart Header, dan kamu letakkan chart di dalamnya. Standar tinggi dari chart adalah 350px, kamu juga bisa mengaturnya dengan menggunakan attribute height pada tag canvas.

Jika kamu belum pernah menggunakan Chart js sebelumnya, saya sarankan untuk membaca dokumentasi resmi dari Chart js!. Disana sudah dijelaskan dengan detail.

Area Chart

Pengunjung, Januari 2020
CopyHTML
<section class="chart">
    <div class="chart__header">
        <h6>Pengunjung, Januari 2020</h6>
    </div>
    <div class="chart__body">
        <canvas id="area_chart"></canvas>
    </div>
</section>
CopyJavascript
// Area Chart
const areaChart = document.querySelector("#area_chart").getContext('2d');
let configAreaChart = {
    type: 'line',
    data: {
        labels: ['Ahad, 11','Senin, 12','Selasa, 13','Rabu, 14','Kamis, 15','Jum\'at, 16'],
        datasets: [{
            label: 'Pengunjung',
            data: [10,6,7,5,1,14],
            fill: 'boundary',
            backgroundColor: 'rgba(37,151,224,.7)',
            borderColor: '#2597e0'
        }]
    },
    options: {
        maintainAspectRatio: false,
        legend: {
            display: false
        }
    }
}
new Chart(areaChart, configAreaChart);

Pie Chart

Pemakaian Browser, Januari 2020
CopyHTML
<section class="chart">
    <div class="chart__header">
        <h6>Pemakaian Browser, Januari 2020</h6>
    </div>
    <div class="chart__body">
        <canvas id="pie_chart"></canvas>
    </div>
</section>
CopyJavascript
// Pie Chart
const pieChart = document.querySelector("#pie_chart").getContext('2d');
let configPieChart = {
    type: 'pie',
    data: {
        labels: ['Chrome','Mozilla','Uc Browser','Opera Mini'],
        datasets: [{
            data: [10,6,7,5],
            backgroundColor: [
                "#1bd741",
                "#2597e0",
                "#f9a022",
                "#dd2525"
            ]
        }]
    },
    options: {
        maintainAspectRatio: false,
        responsive: true,
        legend: {
            display: true
        }
    }
}
new Chart(pieChart, configPieChart);
