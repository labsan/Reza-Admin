Modal

  • Home
  • Modal

Membuat Modal kamu membutuhkan tag div dengan class .modal diikuti dengan class untuk style modal-nya ada .modal--success, .modal--info, .modal--danger, dan .modal--warning. Lalu didalamnya ada tag div dengan class .modal__content, didalamnya lagi ada 3 bagian yaitu, Judul, Deskripsi, dan Aksi.

Pertama, Judul. Bagian judul diisi dengan judul dari Modal-nya, seperti 'Data ditambahkan' atau 'Konfirmasi hapus' dan lain-lain. Membuat judul kamu membutuhkan tag div dengan class .modal__title, tambahkan juga class utilities untuk margin milik-nya bootstrap yaitu .mb-5. Didalamnya kamu bisa menambahkan icon dari fontawesome, dan dibawah icon kamu letakkan judul dari Modal-nya menggunakan tag h3.

Kedua, Deskripsi. Bagian Deskripsi diisi dengan penjelasan detail dari modal-nya, kamu juga bisa menambahkan input jika diperlukan. Membuat Deskripsi kamu membutuhkan tag div dengan class .modal__description, tambahkan juga class .mb-5. Didalamnya kamu bisa menambahkan text menggunakan tag p, atau input.

Ketiga, Aksi. Bagian Aksi diisi dengan button seperti button untuk menutup modal. Membuat Aksi kamu membutuhkan tag div dengan class .modal__action, lalu didalamnya kamu bisa membuat button seperti pada dokumentasi button.

Modal secara standar disembunyikan, kamu bisa menampilkannya dengan menambahkan class .modal--fade-in setelah class untuk style modal atau setelah class .modal. Jika kamu ingin menutup modal-nya maka cukup hapus class .modal--fade-in, itu semua bisa kamu lakukan menggunakan Javascript & DOM.

Tampilkan Modal Success

Data ditambahkan

Selamat, Data-mu berhasil ditambahkan.

Ok, Tutup modal! 
CopyHTML
<a href="#" class="btn btn--green" id="show-modal">Tampilkan Modal Success</a>
<div class="modal modal--success" id="modal-success">
    <div class="modal__content">
    	<div class="modal__title mb-5">
            <span class="fa fa-check mb-2"></span>
    	    <h3>Data ditambahkan</h3>
    	</div>
    	<div class="modal__description mb-5">
    	    <p>Selamat, Data-mu berhasil ditambahkan.</p>
    	</div>
    	<div class="modal__action">
    	    <a href="#" class="btn btn--green" id="close-modal">Ok, Tutup modal!</a>
    	</div>
    </div>
</div><!-- modal-success -->
CopyJavascript
const btnShowModal = document.querySelector("a#show-modal");
const modal = document.querySelector("div#modal-success");
const closeModal = modal.querySelector("a#close-modal");
btnShowModal.addEventListener('click', e => {
    // hilangkan fungsi default dari tag a
    e.preventDefault();

    // tampilkan modal
    modal.classList.add("modal--fade-in");

    /* tambahkan class .stop-scrolling pada tag <body> untuk menghilangkan scroll halaman, 
    agar halaman tidak bisa di scroll */
    document.body.classList.add("stop-scrolling");
});

closeModal.addEventListener('click', e => {
    // hilangkan fungsi default dari tag a
    e.preventDefault();

    // sembunyikan/tutup modal
    modal.classList.remove("modal--fade-in");

    /* hapus class .stop-scrolling pada tag <body> agar 
    halaman bisa di scroll kembali */
    document.body.classList.remove("stop-scrolling");
});
