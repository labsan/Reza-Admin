Footer

  • Home
  • Footer

Membuat footer kamu hanya membutuhkan tag footer dengan class .footer, didalamnya kamu bisa menambahkan text menggunakan tag p, jika kamu punya lebih dari satu tag p, maka tambahkan class .mb-2 .mb-sm-0 pada setiap tag p kecuali tag p yang terakhir. Ini untuk memberi margin bottom kepada tag p ketika di tampilan mobile. Dan yang paling penting adalah kamu letakkan footer setelah Main.
