Halaman 404

  • Home
  • Halaman 404

Membuat halaman 404 (atau Page Not Found atau Halaman Tidak Ditemukan), Pertama kamu bisa copy paste Struktur Dasar Template yang ada dihalaman Instalasi, lalu cocokkan url yang ada dengan struktur aplikasi atau web kamu, dan untuk membuat halaman 404, standarnya kamu tidak perlu meload javascript apapun.

Kedua, didalam tag body buat tag div dengan class .container-fluid. Lalu didalam tag div dengan class .container-fluid buat tag div dengan class .page-not-found, didalamnya kamu buat lagi tag div dengan class .page-not-found__content, didalam tag div dengan class .page-not-found__content ada dua bagian yaitu 404 Head (tempat judul dari halaman) dan 404 Body (tempat keterangan halaman 404).

Untuk membuat 404 Head, kamu perlu tag div dengan class .page-not-found__head, lalu didalamnya kamu bisa menggunakan tag h1 untuk membuat judul.

Untuk membuat 404 Body, kamu perlu tag div dengan class .page-not-found__body, lalu didalamnya kamu bisa menambahkan text apapun untuk keterangan dari halaman 404-nya.

<!DOCTYPE html>
<html lang="en">
<head>
    <!-- Require meta tags -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

    <title>Halaman 404 . Reza Admin</title>

    <!-- pertama Bootstrap CSS -->
    <link rel="stylesheet" href="[host mu]/dist/css/bootstrap.min.css">
    <!-- kemudian Font awesome -->
    <link rel="stylesheet" type="text/css" href="[host mu]/plugins/font-awesome-4.7.0/css/font-awesome.min.css">
    <!-- dan Reza Admin CSS -->
    <link rel="stylesheet" type="text/css" href="[host mu]/dist/css/reza-admin.min.css">
    <!-- Favicon -->
    <link rel="icon" href="[host mu]/dist/img/Reza_logo.ico">
</head>
<body>

    <div class="container-fluid">
        <div class="page-not-found">
            <div class="page-not-found__content">
                <div class="page-not-found__head">
                    <h1>404</h1>
                </div>
                <div class="page-not-found__body">
                    <h5 class="mb-4">Halaman tidak ditemukan.</h5>
                    <a href=""><span class="fa fa-long-arrow-left"></span> Kembali ke dashboard</a>
                </div>
            </div>
        </div>
    </div>

</body>
</html>
