Dasar

Membuat form input kamu perlu menambahkan class .form pada tag input, select atau textarea, diikuti dengan class untuk style form input-nya ketika focus/aktif. Ada class .form--focus-blue, .form--focus-green, .form--focus-red, dan .form--focus-orange.

Jika kamu ingin menambahkan label pada input, kamu perlu menambahkan tag label sebelum input, bungkus label dan input menggunakan tag div dengan class .form-group, supaya script kita lebih rapi.

Kamu juga bisa implementasikan grid sistem bootstrap untuk membuat form dengan kolom. Kamu perlu tag div dengan class .form-row, lalu didalamnya buat tag div dengan class .col-md-6, kamu bisa menyesuaikan ukuran kolomnya, lihat dokumentasi grid sistem bootstrap!, dan kamu letakkan input-nya didalam tag div dengan class .col-md-6.

CopyHTML
<div class="form-row">
    <div class="col-md-6">
        <input type="text" name="nama" placeholder="Nama Depan ..." class="form form--focus-blue mt-0">
    </div>
    <div class="col-md-6">
        <input type="text" name="nama" placeholder="Nama Belakang ..." class="form form--focus-blue mt-3 mt-md-0">
    </div>
</div>
    
<div class="form-group">
    <label>Input label</label>
    <label class="label-right">Wajib</label>
    <input type="text" name="umur" placeholder="Umur ..." class="form form--focus-blue">
</div>

<select name="country" class="form form--focus-blue">
    <option disabled="disabled" selected="selected">Negara ...</option>
    <option>Indonesia</option>
    <option>Palestina</option>
    <option>Arab Saudi</option>
</select>

<textarea name="nama" class="form form--focus-blue" placeholder="Tentang ..."></textarea>   

<input type="file" name="file" class="form form--focus-blue">

Checkbox dan Radio

Membuat Input Checkbox dan Radio kamu membutuhkan tag div dengan class .form-check, didalamnya kamu letakkan input checkbox atau input radio dengan class .form diikuti class untuk style form input-nya ketika focus/aktif, lalu dibawhnya kamu buat label menggunakan tag label dengan class .label-check dan didalam tag label-nya kamu masukkan text keterangan untuk input-nya.

Jika kamu ingin membuat input Checkbox dan Radio tanpa label, kamu perlu menambahkan class .form-check--not-label setelah class .form-check. Jika kamu ingin membuat input Checkbox dan Radio sejajar atau inline kamu perlu menambahkan class .form-check--inline setelah class .form-check.

CopyHTML
<!-- input checkbox dengan label -->
<div class="form-check mt-0">
    <input type="checkbox" name="input_check" id="input_check" class="form form--focus-blue">
    <label for="input_check" class="label-check">Checkbox</label>
</div>

<!-- input checkbox tanpa label -->
<div class="form-check form-check--not-label">
    <input type="checkbox" name="input_check_disabled" class="form form--focus-blue">
</div>

<!-- input radio dengan label -->
<div class="form-check">
    <input type="radio" name="input_radio" id="input_radio" class="form form--focus-blue">
    <label for="input_radio" class="label-check">Radio</label>
</div>

<!-- input radio tanpa label -->
<div class="form-check form-check--not-label">
    <input type="radio" name="input_radio" class="form form--focus-blue">
</div>

<!-- input checkbox dan radio sejajar atau inline -->
<div class="form-check form-check--inline">
    <input type="checkbox" name="input_check" id="input_check" class="form form--focus-blue">
    <label for="input_check" class="label-check">Inline Checkbox</label>
</div>
<div class="form-check form-check--inline">
    <input type="radio" name="input_radio" id="input_radio" class="form form--focus-blue">
    <label for="input_radio" class="label-check">Inline Radio</label>
</div>

Disabled

Membuat input checkbox dan radio disabled kamu perlu menambahkan attribute disabled="" pada tag input, select, textarea, input checkbox, atau input radio.

CopyHTML
<input type="text" name="nama" placeholder="Nama ..." disabled="" class="form form--focus-blue mt-0">

<select name="country" disabled="" class="form form--focus-blue">
    <option>Negara ...</option>
</select>

<textarea class="form form--focus-blue" disabled="" placeholder="Tentang ..."></textarea>

<div class="form-check">
    <input type="checkbox" name="input_check" id="input_check_disabled" class="form form--focus-blue" disabled="">
    <label for="input_check_disabled" class="label-check">Disabled Checkbox</label>
</div>

<div class="form-check">
    <input type="radio" name="input_radio_disabled" id="input_radio_disabled" class="form form--focus-blue" disabled="">
    <label for="input_radio_disabled" class="label-check">Disabled Radio</label>
</div>

Text Bantuan

Membuat text bantuan kamu perlu tag small dengan class .form-text dan .text-muted, jika pada tag input, select, textarea, kamu perlu membungkus input dan text bantuan menggunakan tag div dengan class .form-group dan letakkan text bantuan setelah input, jika pada tag input checkbox dan input radio kamu letakkan text bantuan didalam tag div dengan class .form-check dan letakkan text bantuan setelah label(jika menggunakan label) atau letakkan text bantuan setelah input(jika tidak menggunakan label). Jika kamu membuat input checkbox atau radio tanpa label, kamu tidak perlu menambahkan class .form-text pada tag small.

Pesan Info

Membuat Pesan Info hampir sama dengan membuat Text Bantuan, kamu hanya perlu menambahkan class .form--success, .form--info, .form--warning, atau .form--danger pada tag input, select atau textarea setelah class untuk style form inputnya ketika focus/aktif, dan mengganti class .text-muted pada tag small dengan class .text-green, .text-blue, .text-orange, atau .text-red.

Kamu hanya bisa membuat Pesan Info menggunakan input, select, dan textarea.

Input Addon

Membuat Input Addon kamu perlu tag div dengan class .input-group, didalamnya ada 3 bagian yaitu prepend, (input, textarea, select), dan append. Prepend dan append adalah tempat dimana kamu bisa meletakkan text, input checkbox atau radio, dan button. Prepend disebelah kiri dan Append disebelah kanan. Kamu bisa menggunakan salah satu atau keduanya.

Membuat prepend kamu membutuhkan tag div dengan class .input-group__prepend, didalamnya untuk menambahkan text kamu perlu tag span dengan class .input-group__text dan letakkan text-nya didalam tag span. Untuk menambahkan input(checkbox atau radio) kamu perlu tag div dengan class .input-group__text, lalu didalamnya kamu bisa meletakkan input(checkbox atau radio)-nya. Untuk menambahkan button kamu bisa meletakkan button-nya langsung didalam tag div dengan class .input-group__prepend. Letakkan prepend sebelum input.

Membuat append kamu membutuhkan tag div dengan class .input-group__append, didalamnya untuk menambahkan button, input(checkbox atau radio) dan text caranya sama seperti prepend. Letakkan append setelah input.
