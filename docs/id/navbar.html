Brand

Sama seperti membuat navbar di dokumentasi bootstrap, hanya saja perlu menambahkan class .navbar--white. Jika kamu ingin menggunakan gambar brand, kamu hanya perlu meng-ubah nilai dari attribute width, dan sesuaikan ukurannya.

Navbar 
<nav class="navbar navbar--white">
    <a class="navbar-brand" href="#">
        Navbar
    </a>
</nav>
<nav class="navbar navbar--white">
    <a class="navbar-brand" href="#">
        <img src="[your path]/dist/img/favicon-(80x80).png" width="30" alt="logo reiran">
    </a>
</nav>

Link Navigasi

Kamu bisa membuat link berwarna merah ketika di hover, dengan menambahkan class .nav-link--red-hover. Juga kamu bisa memposisikan link navigasi di sebelah kanan dengan menambahkan class .justify-content-end, lihat dokumentasi flex utilities.

  • About
  • Medsos
  • Home
  • Logout 
<nav class="navbar navbar-expand-sm navbar--white">
    <a class="navbar-brand" href="#"><img src="[your path]/dist/img/favicon-(80x80).png" width="30" alt="logo reiran"></a>
    <button class="navbar-toggler" target="#navbarNav1" type="button">
        <span class="fa fa-navicon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav1">
        <ul class="navbar-nav">
            <li class="nav-item"><a class="nav-link" href="#">About</a></li>
            <li class="nav-item"><a class="nav-link" href="#">Medsos</a>
            </li>
        </ul>
    </div>
</nav>

Icon Notifikasi

Membuat Icon Notifikasi kamu hanya perlu menambahkan class .navbar__notification, pada tag li, dan menambahkan script berikut setelah icon notifikasi-nya.

<span class="navbar__notification-number navbar__notification-number--blue">2</span>
