Unduh

Dari Google Drive

Jika kamu mengunduh dari google drive maka didalammnya terdapat file-file inti dari template, tidak terdapat dokumentasi offline. Sehingga ukuran file nya kecil.

Unduh
Dari Github

Jika kamu mengunduh dari github maka didalamnya terdapat file secara lengkap, termasuk didalamnya dokumentasi offline, dan pastinya ukuran file cukup besar.

Unduh Sumber

Instal

  1. Kamu hanya membutuhkan apa yang ada didalam folder dist dan plugins,
  2. Copy dan paste dua folder itu ke projek kamu,
  3. dan Reza Admin berhasil di instal.

Struktur Dasar Template

Ini adalah struktur dasar template Reza Admin, selebihnya kamu bisa menambahkan sendiri, sesuai dokumentasi.

<!DOCTYPE html>
<html lang="en">
<head>
    <!-- Required meta tags -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

    <title>Dashboard . Reza Admin</title>

    <!-- Bootstrap CSS -->
    <link rel="stylesheet" href="[your path]/dist/css/bootstrap.min.css">
    <!-- Font awesome -->
    <link rel="stylesheet" type="text/css" href="[your path]/plugins/font-awesome-4.7.0/css/font-awesome.min.css">
    <!-- Reza Admin CSS -->
    <link rel="stylesheet" type="text/css" href="[your path]/dist/css/reza-admin.min.css">
    <!-- Favicon -->
    <link rel="icon" href="[your path]/dist/img/Reza_logo.ico">
</head>
<body>

    <h1>Hello World</h1>

    <!-- Optional JavaScript -->
    <!-- jQuery first, then Popper.js, then Bootstrap JS, then Reza Admin JS -->
    <script src="[your path]/dist/js/jquery-3.5.1.slim.min.js"></script>
    <script src="[your path]/dist/js/bootstrap.bundle.min.js"></script>
    <script src="[your path]/dist/js/reza-admin.min.js"></script>
</body>
</html>
