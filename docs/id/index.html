Unduh

Setelah berhasil meng-unduh kamu kamu akan mendapatkan file utama Template Reza Admin, dan dokumentasi versi offline. Keuntungan dari dokumentasi versi offline adalah mungkin ada dari kalian yang tempat tinggalnya di desa, dimana biasanya cukup sulit untuk mendapatkan koneksi internet yang bagus dan stabil setiap saat, dokumentasi offline ini akan cukup membantu. Jika koneksi internet di tempat kamu bagus saya lebih menyarankan kamu untuk melihat dokumentasi versi online, karena akan terus di update, misalnya jika terdapat kesalahan pada dokumentasi atau telah ada fitur baru yang di tambahkan.

Perlu kamu ketahui bahwa Dokumentasi versi offline sesuai dengan versi dari template, jadi setiap versi template akan ada dokumentasi offline yang berbeda. Sedangkan dokumentasi versi online untuk semua versi template.

Instal

  1. Kamu hanya membutuhkan apa yang ada didalam folder dist dan plugins,
  2. Copy dan paste dua folder itu ke projek kamu,
  3. dan Reza Admin berhasil di instal.

Struktur Dasar Template

Ini adalah struktur dasar template Reza Admin, selebihnya kamu bisa menambahkan sendiri, sesuai dokumentasi.

CopyHTML
<!DOCTYPE html>
<html lang="en">
<head>
    <!-- Required meta tags -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

    <title>Dashboard . Reza Admin</title>

    <!-- Bootstrap CSS -->
    <link rel="stylesheet" href="[your path]/dist/css/bootstrap.min.css">
    <!-- Font awesome -->
    <link rel="stylesheet" type="text/css" href="[your path]/plugins/font-awesome-4.7.0/css/font-awesome.min.css">
    <!-- Reza Admin CSS -->
    <link rel="stylesheet" type="text/css" href="[your path]/dist/css/reza-admin.min.css">
    <!-- Favicon -->
    <link rel="icon" href="[your path]/dist/img/Reza_logo.ico">
</head>
<body>

    <h1>Hello World</h1>

    <!-- Optional JavaScript -->
    <!-- jQuery first, then Popper.js, then Bootstrap JS, then Reza Admin JS -->
    <script src="[your path]/dist/js/jquery-3.5.1.slim.min.js"></script>
    <script src="[your path]/dist/js/bootstrap.bundle.min.js"></script>
    <script src="[your path]/dist/js/reza-admin.min.js"></script>
</body>
</html>
