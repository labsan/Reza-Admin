Dasar

Membuat Card kamu membutuhkan tag <section> atau <div> dengan class .card, dan class untuk style Card-nya, ada .card--orange, .card--green, .card--blue, dan .card--red. Letakkan card didalam grid sistem Bootstrap. Didalam Card ada 4 bagian, yaitu Gambar, Label, Deskripsi, dan Aksi.

Pertama, Gambar. Fungsinya sudah pasti untuk tempat gambar dari card, membuatnya kamu memerlukan tag <div> dengan class .card__image, didalamnya gunakan tag <img> dengan class .card-img-top untuk memasukkan gambar.

Kedua, Label. Fungsinya untuk tempat kategori dari card, atau kamu bisa menggunakannya untuk hal yang lain. Membuat Label kamu membutuhkan tag <div> dengan class .card__label, lalu didalamnya kamu bisa meletakkan kategori dari card dengan menggunakan tag <span>. Letakkan Label setelah Gambar.

Ketiga, Deskripsi. Fungsinya untuk tempat seperti nama produk, harga produk, penjelasan singkat mengenai produk, dan keterangan waktu terakhir kali data produk di update. Membuat Deskripsi kamu membutuhkan tag <div> dengan class .card__decription, kemudian didalamnya kamu bisa gunakan tag <h4> untuk nama produk, tag <h5> untuk harga produk, tag <p> untuk penjelasan singkat mengenai produk, dan tag <time> untuk keterangan waktu terakhir kali data produk diupdate. Jangan lupa untuk menambahkan class .mb-3 pada tag <h5> dan tag <p> supaya ada jarak terhadap element dibawahnya. Letakkan Deskripsi setelah Label.

Keempat, Aksi. Fungsi untuk sebagai tempat tombol-tombol aksi seperti hapus dan edit. Membuat Aksi kamu membutuhkan tag <div> dengan class .card__action, lalu didalamnya kamu bisa menggunakan tag <a>, dan didalam tag <a> kamu bisa gunakan tag <span> untuk misalnya meletakkan icon edit. Ada kalanya kamu butuh untuk menampilkan seperti jumlah stock, kamu bisa menambahkan class .card__action--not-link pada tag <a>. Letakkan Aksi setelah Deskripsi.

Loading

Ketika kamu melakukan proses hapus menggunakan tehnik Ajax misalnya, kamu pasti membutuhkan Loading, untuk menandakan bahwa proses hapus sedang berjalan. Membuat Loading sama dengan dokumentasi Loading. Loading kamu letakkan didalam tag <section> atau <div> dengan class .card, sebelum element <div class="card__image">.

Untuk menyembunyikan Loading, kamu hanya perlu menambahkan class utilities Bootstrap untuk display none .d-none setelah class untuk style Loading-nya. Untuk menampilkan Loading kamu bisa menghapus class .d-none. Gunakan Javascript & DOM untuk menghapus atau menambahkan class. Penjelasan detail mengenai Loading dan contoh Loading yang lain, kamu bisa lihat di dokumentasi Loading.
