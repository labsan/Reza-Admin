Dasar

Membuat Card kamu membutuhkan tag section atau div dengan class .card, dan class untuk warna dari Cardnya, ada .card--orange, .card--green, .card--blue, dan .card--red. Letakkan card didalam Grid Sistem bootstrap. Didalam Card ada 4 bagian, yaitu Gambar, Label, Deskripsi, dan Aksi.

Pertama, Gambar. Fungsinya sudah pasti untuk tempat gambar dari card, membuatnya kamu memerlukan tag div dengan class .card__image, didalamnya kamu masukkan tag image dengan class .card-img-top.

Kedua, Label. Fungsi untuk tempat kategori dari card kamu, atau kamu bisa menggunakannya untuk hal yang lain. Membuat card kamu membutuhkan tag div dengan class .card__label, lalu didalamnya kamu bisa meletakkan kategori dari card kamu atau apapun itu dengan menggunakan tag span. Dan yang paling penting letakkan Label setelah Gambar.

Ketiga, Deskripsi. Fungsi untuk tempat seperti nama produk kamu, harga produk kamu, penjelasan singkat mengenai produk kamu, dan keterangan waktu terakhir kali data produk kamu di update. Membuat deskripsi kamu membutuhkan tag div dengan class .card-body dan .card__decription, kemudian didalamnya kamu bisa gunakan tag h3 untuk nama produk, tag h5 untuk harga produk, tag p untuk penjelasan singkat mengenai produk, dan tag time untuk keterangan waktu terakhir kali data produk diupdate. Jangan lupa untuk menambahkan class .mb-3 pada tag h5 dan tag p supaya ada jarak terhadap element dibawahnya dan juga letakkan Deskripsi setelah Label.

Keempat, Aksi. Fungsi untuk sebagai tempat tombol-tombol aksi seperti hapus dan edit. Membuat aksi kamu membutuhkan tag div dengan class .card__action, lalu didalamnya kamu bisa menggunakan tag a. Ada kalanya kamu butuh untuk menampilkan seperti jumlah stock, kamu bisa menambahkan class .card__action--not-link pada tag a. Dan jangan lupa untuk meletakkan Aksi setelah Deskripsi.

Loading

Ketika kamu melakukan proses hapus menggunakan tehnik Ajax misalnya, kamu pasti membutuhkan loading, untuk menandakan bahwa proses hapus sedang berjalan. Membuat loading kamu hanya perlu menambahkan script berikut sebelum script Gambar Card.

CopyHTML
<div class="loading-circle loading-circle--lg loading-circle--blue">
    <div class="loading-circle__fill"></div>
</div>

Untuk menyembunyikan loading, kamu hanya perlu menambahkan class .d-none disebelah class .loading-circle--orange, untuk menampilkan loading kamu bisa menghapus class .d-none. Saya harap kamu sudah mempelajari Javascript sebelumnya. Untuk penjelasan detail mengenai loading dan contoh loading yang lain, kamu bisa baca bagian dokumentasi Loading.
