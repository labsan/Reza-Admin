Dasar

Class utama untuk membuat sidebar:

  • .sidebar digunakan pada tag aside sebagai container dari sidebar, dan membuat posisi sidebar menjadi tetap di kiri browser walaupun halaman di gulir,
  • .sidebar__nav digunakan pada tag ul,
  • .sidebar__item digunakan pada tag li sebagai item-item dari sidebar,
  • sidebar__item--active digunakan pada tag li, sebagai penanda bahwa item itu sedang aktif.
  • Gambaran
  • Rencana Harian
  • Kemajuan
  • Pesan

<aside class="sidebar">
    <ul class="sidebar__nav">
        <li class="sidebar__item">
            <a class="nav-link" href="#"><span class="fa fa-home"></span> Gambaran</a>
        </li>
        <li class="sidebar__item sidebar__item--active">
            <a class="nav-link" href="#"><span class="fa fa-group"></span> Rencana Harian</a>
        </li>

        </li>
        <li class="sidebar__item">
            <a class="nav-link" href="#"><span class="fa fa-envelope"></span> Pesan</a>

    </ul>
</aside>

Dropdown

Membuat menu dropdown, kamu harus menambahkan class .sidebar-btn-dropdown pada tag a, gunanya untuk menandakan bahwa element itu adalah button untuk menampilkan dan menyembunyikan menu dropdown. Kemudian dalam tag a tambahkan script berikut! untuk membuat icon caret ( ).

CopyHTML
<span class="fa fa-caret-right sidebar__span-dropdown"></span>

Selanjutnya untuk membuat badan dari dropdownnya, buat element ul dalam tag li, dengan class .sidebar__dropdown, dalamnya buat item-item dari dropdown dengan tag li class .sidebar__dropdown-item, dan jangan lupakan tag a, letakkan itu dalam tag li seperti sebelumnya, namun bedanya adalah kamu tidak perlu menggunakan class .nav-link. Kamu juga bisa menambahkan class .sidebar__dropdown-item--active pada tag li untuk menandakan bahwa item dropdown itu sedang aktif.

Sidebar dropdown standar-nya disembunyikan. Tapi jika kamu ingin standar sidebar dropdown ditampilkan, kamu perlu menambahkan class .sidebar__item--dropdown-active pada tag li disebelah class .sidebar__item, sebagai penanda bahwa dropdown sedang aktif. Kemudian tambahkan class .sidebar__span-dropdown--rotate untuk memutar icon caret 90 derajat dan yang paling penting kamu tambahkan class .sidebar__dropdown--show disebelah class .sidebar__dropdown, untuk mengubah standar sidebar dropdown menjadi ditampilkan.

Toggler

Standarnya sidebar akan disembunyikan jika lebar window <= dari 767.98px. Membuat toggler sidebar kamu hanya perlu menambahkan script berikut pada navbar, letakkan sebelum script Navbar Brand!

CopyHTML
<a class="navbar__sidebar-toggler" href="#"><span class="fa fa-bars"></span></a>
